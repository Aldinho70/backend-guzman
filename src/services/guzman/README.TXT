La operacion de guzman se basa en estos apartados:

    Una unidad maneja su estatus de operacion de la siguiente manera:
    ----------------------------------------------------------------------------------
    Unidad
        -> Cargadas
            -> Unidades cargadas (Resumen general de todas las unidades)
            -> Unidades cargadas detenidas
            -> Unidades cargadas en movimiento
            -> Unidades cargadas sin reportar
            -> Unidades cargadas desviadas
        -> Vacia
            -> Unidades vacias ( Resumen general de todas las unidades )
            -> Unidades vacias en movimiento
            -> Unidades vacias detenidas
        -> Descargando
            -> Unidades descargando ( Resumen general de todas las unidades )
        -> Cargando
            -> Unidades cargando ( Resumen general de todas las unidades )
        -> Sin estatus
            -> Unidades sin estatus
    ---------------------------------------------------------------------------------
    Cajas
        -> Cajas sin reportar
    Cajas dobles
        -> Cajas dobles sin reportar
    ---------------------------------------------------------------------------------
    Tractos
        -> Principales sin reportar
    Tractos Dobles
        -> Dobles sin reportar
    ---------------------------------------------------------------------------------
    Temperatura
        -> Falla de temperatura
    ---------------------------------------------------------------------------------

[IRAPUATO TRACTOS GUZMAN]
    -> Tractos    

[DEV GUZMAN TRACTOS DOBLES]
    -> Tractos dobles

[GUZMAN IRAPUATO CAJAS]
    -> Cajas

[GUZMAN CAJAS DOBLES]
    -> Cajas dobles
